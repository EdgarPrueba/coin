COIN_DLL_API void* cc_sync_begin ( void *  id)

Begin synchronizing id. After returning from this call, other threads trying to synchronize id will be blocked until the first thread calls cc_sync_end() with the key returned from this function.

◆ cc_sync_end()

COIN_DLL_API void cc_sync_end ( void *  key)

End synchronize. key should be the key returned from cc_sync_begin().

◆ cc_sync_free()

COIN_DLL_API void cc_sync_free ( void *  id)

Free sync item. This can be called from the destructor of objects that use a sync item so that the sync-mutex is freed when the object is destructed.
