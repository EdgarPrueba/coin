◆ cc_heap_construct()

COIN_DLL_API cc_heap* cc_heap_construct ( unsigned int  size,
cc_heap_compare_cb *  comparecb,
SbBool  support_remove 
)

Construct a heap. size is the initial array size.

For a minimum heap comparecb should return 1 if the first element is less than the second, zero if they are equal or the first element is greater than the second. For a maximum heap comparecb should return 1 if the first element is greater than the second, zero if they are equal or the first element is less than the second.

support_remove specifies if the heap should support removal of elements (other than the top element) after they are added; this requires use of a hash table to be efficient, but as a slight runtime overhead will be incurred for the add and extract_top functions the support can be disabled if you don't need it.
